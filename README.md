# 💰 Python Expense Tracker

A simple Python program that helps users track their daily expenses.  
Users can add expenses, view them, and calculate the total amount spent.

## 🚀 Features
- Add new expenses
- View all expenses
- Calculate total spending
- Simple menu-driven interface
- Beginner-friendly project

## 🛠 Built With
- Python

## ▶️ How to Run

1. Download or clone the repository  
2. Open the project folder  
3. Run the program  

python expense_tracker.py
